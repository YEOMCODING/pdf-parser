# PRD: Python PDF Parser

## 목표

Python 기반 `pdf-parser`를 CLI와 라이브러리 형태로 제공하여 PDF 문서의 텍스트, 표, 메타데이터를 구조화 JSON으로 안정적으로 추출한다.

## 사용자

- PDF 문서를 후처리 가능한 데이터로 변환해야 하는 개발자
- 문서 분석, 검색, RAG 파이프라인에 PDF 내용을 입력하려는 사용자
- 표와 본문 텍스트를 자동 추출해 검수/가공하려는 데이터 작업자

## 핵심 기능

1. CLI에서 PDF 파일을 입력받아 JSON을 stdout 또는 파일로 출력한다.
2. Python API `parse_pdf(path, options=None)`를 제공하여 다른 애플리케이션에서 라이브러리로 사용할 수 있게 한다.
3. PyMuPDF로 문서 메타데이터, 페이지 정보, 텍스트 블록을 추출한다.
4. pdfplumber로 페이지별 표를 추출하고 행/열 구조를 JSON에 보존한다.
5. 텍스트가 거의 없는 스캔형 PDF를 감지하고 OCR 제외 warning을 남긴다.
6. 파싱 실패와 부분 실패를 명확한 예외 또는 warning으로 표현한다.

## MVP 제외 사항

- OCR 기반 스캔 PDF 텍스트 인식
- 웹 UI, 데스크톱 GUI, HTTP API 서버
- 단어 단위 좌표와 원본 레이아웃 완전 재구성
- 대용량 병렬 처리와 성능 최적화
- PDF 편집, 병합, 분할, 주석 처리 기능

## 성공 기준

- 텍스트 PDF에서 페이지 순서와 본문 텍스트가 안정적으로 추출된다.
- 표가 포함된 PDF에서 기본적인 행/열 구조가 JSON으로 출력된다.
- 스캔형 PDF는 실패처럼 보이지 않고 명시적인 warning을 제공한다.
- CLI와 Python API가 같은 파싱 엔진과 같은 JSON 스키마를 사용한다.
- 테스트는 정확성을 우선 검증하며, 모든 구현은 TDD 순서로 진행한다.
